# Welcome to Your New Blog!

Congratulations! Your blog is now live on the internet. This is your first post - feel free to edit or delete it.

## Getting Started

### Customize Your Blog

You have a few options to make this blog yours:

**Option 1: Edit in Tina CMS (Visual Editor)**
1. Go to [app.tina.io](https://app.tina.io) and sign in
2. Select your blog project
3. Click "Edit with Tina"
4. Update Site Settings: name, bio, colors, social links
5. Edit or delete this post
6. Create your own posts!

**Option 2: Edit on GitHub (Simple)**
1. Go to your repository on GitHub
2. Navigate to `content/settings/site.json`
3. Click the pencil icon to edit
4. Update your information
5. Commit changes

### Write Your First Real Post

Create a new file in `content/posts/` with this format:

```markdown
---
slug: "my-first-post"
title: "My First Real Post"
date: 2025-11-03
author: Your Name
tags:
  - Personal
description: "This is my first blog post!"
---

# My First Post

Write your content here using Markdown!

## You can add headings

- Bullet points
- **Bold text**
- *Italic text*
- [Links](https://example.com)

![Images](/images/your-image.png)
```

### Add Images

Upload images to `public/images/` in your repository, then reference them in posts:

```markdown
![Image description](/images/your-image.png)
```

## What You Can Customize

Through Site Settings (edit `content/settings/site.json`):
- Your name and bio
- Site title and description
- Theme colors (primary, background, text, accent)
- Social media links (Twitter, LinkedIn, GitHub)
- SEO keywords

---

## Tips for Great Blog Posts

1. **Write compelling titles** - They appear in search results
2. **Add clear descriptions** - Help readers know what to expect
3. **Use tags** - Organize your content by topic
4. **Add images** - Make posts more engaging
5. **Schedule posts** - Set future dates to publish later
